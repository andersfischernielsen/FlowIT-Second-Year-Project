# JSON Objects
POST to Server:
```json
{
    "Name": "Workflow Test 1",
    "Id": "test1"
}
```

POST to Event:

```json
{
    "EventId": "test1",
    "WorkflowId": "test1",
    "Name": "Test 1",
    "Pending": false,
    "Executed": false,
    "Included": true,
    "Conditions": [],
    "Exclusions": [],
    "Responses": [],
    "Inclusions": []
}
```
